# SQL-Business-Queries

## Project Description
This repository contains SQL queries written to solve real-world business and interview-style problems.  
The focus of this project is on applying SQL concepts to analyze data, answer business questions, and derive insights from relational databases.

All queries are written using structured logic and standard SQL practices.

---

## Objectives
- Practice SQL using business-oriented questions
- Strengthen understanding of relational databases
- Improve problem-solving and analytical thinking
- Build a SQL portfolio for interviews

## Tools Used
- SQL
- Relational Database Concepts
- GitHub

## SQL Topics Covered
- Filtering and Sorting
- Aggregations (SUM, COUNT, AVG)
- GROUP BY and HAVING
- Joins (INNER, LEFT)
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Ranking Functions
- Data Analysis using SQL

## Problem Statements Included
- Identify senior-most employees
- Analyze invoice data by country and city
- Find best customers based on spending
- Identify top artists and genres
- Analyze customer spending on artists
- Determine most popular genres by country
- Compare track duration with average length
